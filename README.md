# delinquency-prediction-project
Customer Delinquency Prediction Project using Python, EDA, and Machine Learning techniques.

## Project Overview
This project focuses on Exploratory Data Analysis (EDA) of customer financial and behavioral data to identify delinquency risk factors.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Dataset Features
- Age
- Income
- Credit Score
- Credit Utilization
- Missed Payments
- Loan Balance
- Debt-to-Income Ratio
- Employment Status

## Project Goal
To analyze customer financial behavior and prepare data for predictive machine learning models.
